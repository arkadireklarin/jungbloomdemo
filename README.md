# Jungbloom Demo

Jungbloom is an explainable AI assignment platform designed to diagnose learning friction from student interaction signals and adapt follow-up questions accordingly.

This repository contains the public demo interface of Jungbloom. The core diagnostic logic, scoring rules, and adaptive generation engine are kept private.

## What this demo shows

- Teacher worksheet upload flow
- Student solving workspace
- Interaction signal tracking UI
- Learning friction explanation cards
- Adaptive follow-up question preview

## Core idea

Most assignment platforms track whether a student is right or wrong. Jungbloom focuses on why the student is stuck.

It observes learning behavior such as hesitation, retries, skips, response time, and error patterns, then explains what it noticed and how the next question is adapted.
